# My Tableau Project(24030105031)

## Description
This repository contains Tableau dashboards and workbooks for the purpose of the analysis of The Global Food System Diet, Health, and Inequality.

## Files
- `The Global Food System Diet, Health, and Inequality.twb` - tableau project
- `The Global Food System Diet, Health, and Inequality.xlsx` - data utilised

## How to Use
1. Download the `.twb` files
2. Open in Tableau Desktop
3. Connect to provided data source

## Notes
- Use `.twb` format for version control
- Do not commit `.twbx` files with sensitive data
